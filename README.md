# Build AI

A modern AI-powered application built with **Next.js**, **TypeScript**, and **Tailwind CSS**. This project features user authentication, database integration, and real-time UI components.

## Features

- 🔐 **User Authentication** - Secure login with Clerk
- 🗄️ **Database** - PostgreSQL with Prisma ORM
- 🎨 **UI Components** - Built with shadcn/ui and Tailwind CSS
- 🔒 **Security** - Arcjet protection against common web attacks
- 🌙 **Dark Mode** - Theme support with next-themes
- ⚡ **Performance** - Optimized Next.js 16 with React 19

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL database
- Environment variables configured

### Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL + Prisma
- **Authentication**: Clerk
- **Security**: Arcjet
- **UI Library**: shadcn/ui
- **Icons**: Lucide React

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
